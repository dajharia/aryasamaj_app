# जबलपुर आर्यसमाज ERP
 
आर्यसमाज मंदिर के लिए एकीकृत प्रबंधन प्रणाली
 
## Features
 
- **पुस्तकालय प्रबंधन** - किताबों की इन्वेंटरी, सदस्यता, जारी/वापसी
- **विवाह मंडप बुकिंग** - बुकिंग स्लॉट, भुगतान, उपलब्धता कैलेंडर
- **कैटरिंग सेवा** - मेनू प्रबंधन, ऑर्डर, भुगतान
- **रेसिडेंशियल प्रबंधन** - कमरा बुकिंग, चेक-इन/चेक-आउट
- **सदस्य प्रबंधन** - सदस्य रजिस्ट्रेशन, दान, सदस्यता शुल्क
- **वित्तीय प्रबंधन** - आय/व्यय, रिपोर्ट, लेखा
- **AI Integration** - स्मार्ट रिपोर्टिंग और ऑटोमेशन
 
## Tech Stack
 
- **Next.js 14+** - Modern React framework
- **TypeScript** - Type safety
- **TailwindCSS** - Modern styling
- **shadcn/ui** - Beautiful UI components
- **Prisma + SQLite** - Offline-first database
- **PWA** - Progressive Web App for offline support
 
## Installation
 
```bash
npm install
```
 
## Database Setup
 
```bash
npx prisma generate
npx prisma db push
```
 
## Development
 
```bash
npm run dev
```
 
## Build
 
```bash
npm run build
npm start
```
